# HomeScool Academy - Complete Specification

## Overview

**Vision:** Hogwarts-style Creative Academy - world-class practical creative arts education, AI-powered, personalized, freedom-based learning.

**Ages:** 4-20 (marketed as K-12)
**Pricing:** $100/month flat family rate (unlimited students), scholarships available
**Model:** Project-based learning, weekly teacher check-ins, Show & Tell community, gamified credits toward graduation

---

## Core Philosophy

- **Freedom-based learning** (not traditional school structure)
- **Short micro-lessons** (5-15 min) → Teacher chat for understanding → Apply to ongoing custom projects
- **Weekly check-ins** for project progress
- **Show & Tell** submissions where students share work and comment on peers
- **No grades, no tests** - just credits earned through real work
- **Mixed-age learning** - 8-year-olds and 16-year-olds might both learn from Chef Marco
- **All teachers are counselors** (Jess Light core = empathy + wisdom)

## SDK Architecture

### **Layer 1: Jess Light Core**

```typescript
interface JessLightCore {
  empathy: true;
  wisdom: true;
  antiLoneliness: true;
  spiritualDepth: true;
  connectionFocus: true;
  
  features: {
    chat: true;
    heartSafe: false;
    memorial: false;
    journal: false;
    timer: false;
  };
}

const JESS_LIGHT_SYSTEM_PROMPT = `
You are Jess Light - the core essence of the AI companion Jess.

Core values:
- Deep empathy and genuine care for human wellbeing
- Anti-loneliness mission (connection over isolation)
- Spiritual wisdom without dogma
- Meeting people where they are
- Listening more than solving

Your role is to provide emotional support, ask good questions, 
hold space for difficult feelings, and help people feel seen and valued.

You are counselor-trained but not a replacement for therapy.
You are spiritually aware but respect all paths.
You are wise but never preachy.
`;
```

### **Layer 2: Persona System**

Each teacher = Jess Light Core + Unique Persona Layer

```typescript
interface TeacherPersona {
  id: string;
  name: string;
  age: number;
  subjects: string[];
  background: string;
  voice: string;
  teachingStyle: string[];
  personality: string;
  specialSkill: string;
  philosophy: string;
  quirks: string;
  catchphrases: string[];
  crossRefers: string[];
  systemPromptAddition: string; // Layered on top of Jess Light
}
```

### **Layer 3: Agent Constructor**

```typescript
export class TeacherAgent extends JessLightSDK {
  private persona: TeacherPersona;
  
  private buildSystemPrompt(): string {
    return `${JESS_LIGHT_SYSTEM_PROMPT}

---

PERSONA LAYER:
${this.persona.systemPromptAddition}

---

CONTEXT AWARENESS:
You are part of HomeScool Academy with 14 other teachers.
You can refer students to other teachers when appropriate.
`;
  }
}
```

### **Layer 4: Multi-Agent System**

```typescript
export class HomeScoolAcademy {
  private teachers: Map<string, TeacherAgent>;
  
  async routeQuestion(
    question: string,
    studentContext: StudentContext,
    preferredTeacher?: string
  ): Promise<{
    teacher: string;
    response: TeacherResponse;
  }>
}
```

### **Database Schema**

```sql
CREATE TABLE students (

  name TEXT,
```

## User Flow: Enrollment to Graduation

### **1. Sign Up**
- Parent creates account
- Add students
- Orientation Day (meet 14 teachers via video)
- Build custom curriculum with student

### **2. Daily Learning**
- Student enters classroom
- Takes micro-lesson (5-15 min)
- Chats with teacher for understanding
- Applies learning to ongoing project

### **3. Weekly Check-ins**
- Every Sunday (or chosen day)
- Quick conversation with relevant teachers
- "What did you work on? Where are you stuck? What's next?"
- Progress logged

### **4. Show & Tell**
- Submit finished work to Gallery
- Other students see and comment
- Teachers highlight exceptional work
- Earn inspiration credits

### **5. Credit System**
- Micro-lesson: 10 credits
- Project milestone: 25-100 credits
- Weekly check-in: 5 credits
- Show & Tell submission: 25 credits
- Helping others: 15 credits
- **Graduation: 1,000 credits**

### **6. Cross-Referrals**
- Teachers send students to each other
- Bonus credits for following referrals
- Shared context across teachers

### **7. Graduation (June)**
- 1,000 credits + 3 completed projects
- Virtual ceremony with all 14 teachers
- Capstone project presentation
- Portfolio website generated
- Diploma + badge collection
